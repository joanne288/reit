# 🏢 REIT Investment Platform (Frontend)

A responsive web application that simulates a Real Estate Investment Trust (REIT) platform, enabling users to explore real estate properties and understand investment opportunities through an interactive interface.

## 🚀 Features

- 🔍 Browse and explore real estate properties
- 📊 View property details and basic investment insights
- 💰 Simulated investment calculations
- 🎨 Responsive and modern UI design
- ⚡ Fast and smooth user experience

## 🛠️ Tech Stack

- React.js
- HTML5
- CSS3
- JavaScript (ES6+)

## 📂 Project Structure

```

reit/
│── src/
│   ├── components/     # Reusable UI components
│   ├── pages/          # Application pages
│   ├── assets/         # Images and static files
│   ├── App.js
│   ├── index.js
│── public/
│── package.json
│── README.md

````

## ⚙️ Installation & Setup

1. Clone the repository:
```bash
git clone https://github.com/joanne288/reit.git
cd reit
````

2. Install dependencies:

```bash
npm install
```

3. Run the application:

```bash
npm start
```

4. Open in browser:

```
http://localhost:3000
```

## 📊 Project Overview

This project demonstrates a frontend-based simulation of a REIT platform, focusing on UI design, component-based architecture, and basic investment logic without backend integration.
